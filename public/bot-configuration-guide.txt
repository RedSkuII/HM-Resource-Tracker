====================================================================
BOT BEHAVIOR TOGGLES
====================================================================

‚úÖ Auto-Update Discord Embeds
   When enabled, the bot automatically updates stock embeds in Discord 
   whenever resources change on the website. Disable this if you prefer 
   manual updates.

‚úÖ Notify on Website Resource Changes
   When enabled, the bot sends notifications to bot channels when 
   resources are updated via the website. Useful for keeping the 
   community informed.

‚úÖ Allow Public Orders
   When enabled, any server member can place orders. When disabled, 
   only users with admin roles can place orders (more restricted mode).

====================================================================
WORKFLOW FOR SERVER OWNERS
====================================================================

1. Select your Discord server from the dropdown (bot must be installed)
2. Link it to an in-game guild (the database that stores resources)
3. Choose bot channels where inventory updates will be posted
4. Choose order channels where members can request resources
5. Select admin roles that can manage the bot configuration
6. Set the resource update bonus percentage (0-200%)
7. Configure notification preferences with the three toggles
8. Click Save Configuration

====================================================================

‚ùå Can't see my Discord server in the list?
   Make sure you've granted the app permission to see your servers 
   when logging in. Try logging out and back in, and check the OAuth 
   permissions.

‚ö†Ô∏è Server shows "Bot needs to be added to this server"?
   The bot isn't installed on that server. Only servers where the bot 
   is installed can be configured. Add the bot to your server first.

üîÑ Changes not saving?
   Check the browser console for errors. Make sure you've selected at 
   least one in-game guild. Verify you have admin permissions on the 
   Discord server.

üìä Bot not posting updates?
   Ensure "Auto-Update Discord Embeds" is enabled. Check that the bot 
   has permissions to post in the selected bot channels. Verify the 
   channel IDs are correct.

API ENDPOINTS FOR BOT DEVELOPERS
====================================================================

GET /api/bot/config/{guildId}
   Retrieve bot configuration for a specific Discord server

PUT /api/bot/config/{guildId}
   Update bot configuration (requires authentication)

GET /api/discord/user-servers
   Get list of Discord servers the authenticated user manages

GET /api/discord/bot-status/{guildId}
   Check if the bot is present in a Discord server
